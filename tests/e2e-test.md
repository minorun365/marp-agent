# E2Eテストチェックリスト

## テスト環境
- URL: http://localhost:5173
- スクリーンショット保存先: tests/screenshots/

## テストケース

### TC1: 認証画面表示
- [ ] ページタイトル「パワポ作るマン」が表示される
- [ ] サインイン画面が日本語で表示される
- [ ] コンソールエラーがない

### TC2: ログイン後のメイン画面
- [ ] チャットUIが表示される
- [ ] ヘッダーにログアウトボタンがある
- [ ] 初期メッセージがアニメーション表示される

### TC3: スライド生成（基本）
- [ ] メッセージを入力して送信できる
- [ ] 「考え中...」ステータスが表示される
- [ ] 「スライドを生成中...」ステータスが1つだけ表示される
- [ ] 「スライドを生成しました」に更新される（チェックマーク付き）
- [ ] プレビュータブでスライドが表示される
- [ ] スライドにborderテーマが適用されている（グレー枠線＋グラデーション背景）

### TC4: Web検索付きスライド生成
- [ ] 「〇〇について調べてスライドを作って」と入力
- [ ] 「Web検索中...」ステータスが表示される
- [ ] 「Web検索完了」に更新される
- [ ] 「スライドを生成中...」→「スライドを生成しました」と遷移

### TC5: PDFダウンロード
- [ ] プレビュー画面で「PDFをダウンロード」ボタンをクリック
- [ ] PDFがダウンロードされる
- [ ] 日本語が正しく表示される（文字化けなし）

### TC6: ステータス重複なし
- [ ] 各ステータスメッセージが1つだけ表示される
- [ ] 複数回のtool_useイベントでも重複しない

## 実行コマンド
```bash
# devサーバー起動
npm run dev

# sandbox起動（別ターミナル）
export TAVILY_API_KEY=$(grep TAVILY_API_KEY .env | cut -d= -f2) && npx ampx sandbox
```
